# ar-maps-pano

## Getting Started

To get started:

```bash
npm i
npm run dev
```

BrowserSync should open http://localhost:4200. To test on your phone, check your local network IP (http://192.168.x.x:4200).

## Create stand

In other terminal tab

```bash
npm i -g localtunnel
lt --port 4200
```

Now your can use this link on mobile or share with other people.
